# imgAspect

一个纯前端、单文件的图片比例换算工具。把设计稿里的图片切段、标注、换算到目标宽度后，能直接看到每一段的原始像素、换算像素、百分比和 `rem` 值，适合做移动端切图和前端布局对照。

## 功能

- 支持上传图片，点击、拖拽、粘贴都可以
- 自动读取原始尺寸、比例和文件大小
- 支持设置目标宽度和 `rem` 基准
- 自动生成图片边界线，方便分段计算
- 可拖拽分割线，实时调整每一段数据
- 数据面板会显示每段的原始 `px`、换算 `px`、占比和 `rem`
- 支持删除用户自定义分割线
- 支持裁切下载
- 深色 / 浅色主题切换

## 适合谁用

- 前端开发
- UI 还原
- 移动端适配
- 设计稿尺寸换算
- 需要把图片拆段做像素参考的人

## 核心公式

```text
scale = targetWidth / naturalWidth
convertedPx = originalPx × scale
percent = convertedPx / convertedDimension × 100
rem = convertedPx / remBase
```

## 使用方式

直接在浏览器里打开 `index.html` 即可使用，无需安装依赖。

## 项目特点

- 单文件实现，打开即用
- 无第三方运行依赖
- 通过 SVG 叠加层绘制分割线和标签
- 分割线位置使用原始像素坐标保存，避免缩放后失真
- 边界线固定存在，不可拖拽删除
